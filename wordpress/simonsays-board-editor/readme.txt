Stateless in-browser editor for SimonSays communication boards (.ssc / .ssz) with live preview, templates and ARASAAC pictogram search.

== Description ==

SimonSays Board Editor lets visitors create, import, edit, preview, validate and export communication boards for the SimonSays – Simply Speak desktop AAC application, directly in their browser.

* **Stateless by design** — boards are opened from and saved to the visitor's own device. Board content is never uploaded to, processed by, or stored on this website. An in-browser autosave (IndexedDB) protects against accidental tab closes.
* **Live preview** — the board is rendered as the SimonSays category window would show it: style cascade, colors, icons, caption strip, and right-to-left mirroring for RTL board languages.
* **Full style editing** — every board-style and category-style property, plus a raw style-list field. Style properties from newer SimonSays versions are preserved verbatim.
* **Templates** — start from a blank board or the bundled SPC (color-coded) example. Site administrators can upload, rename and delete additional templates (each tagged with a board language) from the settings page.
* **ARASAAC pictograms** — search the official ARASAAC API and insert pictograms as icons. Attribution is always displayed and added to the board credits.
* **Safe imports** — untrusted .ssz bundles are validated (structure, entry names, size and compression limits, content sniffing) before anything is used.
* **Accessible** — fully keyboard-operable with visible focus and labelled controls.

Embed the editor with the "SimonSays Board Editor" block or the `[simonsays_board_editor]` shortcode. Settings (Settings → SimonSays Boards) control exposure, size limits, templates, and the ARASAAC integration.

== Privacy ==

Board content never leaves the visitor's browser. The only outbound network traffic generated by the editor is the pictogram search: when a visitor uses the ARASAAC panel, their search keywords and locale are sent directly from their browser to the ARASAAC API (api.arasaac.org / static.arasaac.org). No other personal data is collected, transmitted, or stored by this plugin.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` and activate it.
2. Complete (or skip) the ARASAAC license affirmation shown after activation.
3. Add the "SimonSays Board Editor" block or the `[simonsays_board_editor]` shortcode to a page.
4. Review Settings → SimonSays Boards (exposure, size limits, default template).

== Frequently Asked Questions ==

= Where are the boards stored? =

Nowhere on the server. Visitors open `.ssc`/`.ssz` files from their device and download the edited result. A browser-local autosave protects unsaved work.

= Which file formats are supported? =

`.ssc` (plain text) and `.ssz` (a Zip bundle with icons and sounds), exactly as produced and consumed by the SimonSays desktop application. Resources are limited to `.ico`, `.png`, `.jpg`, `.wav` and `.mp3`.

= Does it work without ARASAAC? =

Yes. The pictogram panel is optional and disabled until the license affirmation is completed; everything else works independently.
